Bienvenu sur ce petit outil.

Pour s'en servir :

*Lancer l'executable (evidemment)

*Ecrire le path complet de l'image (du type C:\Users\Renaud\Desktop\source_dir\0.png) dans le champ prévu à cet effet

*Cliquer sur l'image chargée pour créer des points (/!\ les 2 premiers points forment une diagonale)

*Click-and-drag des points pour les déplacer

*Scroll pour augmenter/réduire le rectangle.

*Bouton "recommencer" pour repartir de zéro sur la même image

*Bouton "finir" pour enregistrer la hitbox
